# Upload instruction

### Specification .bin files
```
  -  Module: Generic ESP8266 Module
  -  Flash Size: 1M
  -  CPU Frequency: 160Mhz
  -  Flash Mode: dio
  -  Flash Frequency: 80Mhz
  -  Upload Speed: 921600
```
