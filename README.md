## Or NodeMCU Flasher:
Specification [esp8266-Radio.ino.generic.bin](https://github.com/renat2985/esp8266-Radio/raw/master/esp8266-Radio.ino.generic.bin) files
```
  -  Module: Generic ESP8266 Module
  -  Flash Size: 1M
  -  Flash Frequency: 80Mhz
```
